Cifrado de Contraseñas con BCrypt

En Gestor de Red utilizamos BCrypt, un algoritmo de hash criptográfico de última generación para proteger tus contraseñas.

¿Qué es BCrypt?

BCrypt es un algoritmo de hash adaptativo que fue diseñado específicamente para el cifrado de contraseñas. A diferencia de otros métodos, BCrypt es:

  • Lento intencionalmente: Hace que sea computacionalmente costoso romper contraseñas mediante fuerza bruta
  • Adaptativo: Su coste computacional puede aumentarse con el tiempo conforme los procesadores se vuelven más rápidos
  • Con "salt": Añade un componente aleatorio a cada hash, lo que evita ataques de tabla arcoíris
  • Unidireccional: No se puede desencriptar, solo verificar

¿Cómo funciona?

1

Registro

Introduces tu contraseña en texto plano

2

Hasheo con BCrypt

BCrypt genera un hash único e irreversible

3

Almacenamiento

Se guarda SOLO el hash en la BD, nunca la contraseña

En el Login

1

Introduces tu contraseña

Se envía de forma segura (HTTPS)

2

Se valida con BCrypt

La contraseña se compara con el hash guardado

3

Acceso concedido

Si coincide, creas tu sesión

Medidas de Seguridad Adicionales

HTTPS Seguro

Todas las conexiones están cifradas

Sin Reversibilidad

Incluso los administradores no pueden ver tus contraseñas

Salt Único

Cada contraseña tiene un componente aleatorio único

Validación en Servidor

La verificación se realiza siempre en el servidor, nunca en el cliente

Recomendaciones de Seguridad

Contraseña Fuerte

Usa una combinación de mayúsculas, minúsculas, números y caracteres especiales

No Compartas tu Contraseña

Nunca compartas tus credenciales con nadie, ni siquiera con soporte técnico

Cambia Regularmente

Actualiza tu contraseña cada 3 meses por seguridad

Cierra la Sesión

Siempre cierra sesión cuando termines de usar la aplicación
