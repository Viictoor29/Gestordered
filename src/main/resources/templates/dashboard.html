Modificar topología

Añadir host

Usa hosts H1-H255. La MAC se genera automaticamente con ese numero y la IP se valida contra la red de los hosts existentes.

Añadir switch
Añadir enlace

En origen y destino puedes indicar un host o un switch. Se admiten enlaces host-switch y switch-switch. Los puertos se calculan automaticamente con el primer puerto libre.

Control de bucles

Estado STP

STP evita bucles bloqueando algunos puertos. Tras cambios en la topologia, puede tardar unos segundos en converger.
Conecta primero con la API para consultar el estado STP.

Filtros activos

IPs bloqueadas

Conecta primero con la API para consultar las IPs bloqueadas.

Pruebas activas

Generar trafico

Estas pruebas generan trafico real entre hosts de Mininet. Si STP acaba de converger, espera unos segundos antes de lanzar pruebas intensivas.
Conecta primero con la API para generar trafico.

Diagnostico

Metricas de salud y flujos

Las metricas dependen de las estadisticas que Ryu recibe de los switches. Tras cambios recientes, pueden tardar unos segundos en estabilizarse.
Conecta primero con la API para consultar la salud de la red.
